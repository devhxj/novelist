# Release Notes

### 2026-07-05

- 创建参考锚点时可通过系统文件选择器选择本地参考源文件，仍保留手动输入路径能力。
- 选择 `.txt`、`.md` 或 `.markdown` 文件后，参考源格式会按扩展名自动匹配，减少导入时的手动切换。

### 2026-07-04

- 新增参考锚定工作区入口，可建立参考素材、搜索素材，并按蓝图评审、素材绑定、候选草稿审核的顺序推进。
- 参考锚定蓝图现在可在工作区内修订事实、POV、情绪证据、段落意图、正文职责和引用查询；保存后会回到待评审状态，必须重新评审并批准后才能继续绑定材料或生成候选段落。
- 蓝图评审和候选草稿审计改为分类展示，可直接查看逻辑、因果、情绪、POV、材料匹配、剧本化风险和必须修复项。
- 材料绑定会显示评分组成，并只允许使用匹配当前蓝图分析合同的有效材料链接生成候选段落。
- 候选段落预览会展示来源材料、节拍、改写等级、变更槽位、非槽位改动和审计风险。
- Agent 可调用参考素材检索、改写审计和蓝图驱动草稿工具，但不会通过该流程直接写入章节正文。
- Novelist 桌面版启动更直接，不再依赖本机 ASP.NET/Kestrel 服务作为产品运行时，安装包启动路径更轻量。
- 封面显示改为通过桌面桥接读取，打包后的应用不再需要 `/covers` HTTP 路由也能显示小说封面。
- 开发模式仍可通过 Vite 地址调试前端，桌面业务调用继续走 Photino bridge。
